h1. Textile Text Formatter for Symphony CMS

This extension provides "Textile":https://github.com/textile/php-textile formatting for fields.

h3. Synopsis

Format text using "Textile syntax":https://txstyle.org/.

h3. Installation & Updating

Information about "installing and updating extensions":http://getsymphony.com/learn/tasks/view/install-an-extension/ can be found in the "Symphony documentation":http://getsymphony.com/learn/.

# Clone/Download this repo.
# Unzip and rename resulting folder to @textile@. This step is very important!
# Upload @textile@ folder to your symphony extenstions folder.
# In the Symphony backend System | Extensions area, select the Textile row and choose Install from the options dropdown.
# When you add a "Textarea" field to any of your Sections you can now choose "Textile" as a Text Formatter.
# Enjoy the most advanced and superior Text Formatter on planet.
